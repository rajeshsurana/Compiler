Welcome to the Compiler wiki!

This compiler is a result of course CSE340 project. It is completely error free and robust! Let's start checking it now! This is the compiler for Java like language and does not detect some of the Java constructs like "for". But, it almost covers all the constructs. In the Parser Requirement document you can check out the supported constructs.
